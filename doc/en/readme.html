Overview

This module improves accessibility of the Notepad++ editor for blind or visually impaired developers using the NVDA screen reader. It adds keyboard shortcuts to navigate through Python and C/C++ source code without reading the file line by line.

Compatibility:

  • Notepad++ 32-bit and 64-bit (Windows 10 / Windows 11)
  • NVDA 2025.1 minimum, tested up to NVDA 2026.1
  • Supported languages:
    • Python: .py .pyw .pyi
    • C/C++/Arduino: .c .cpp .cxx .h .hpp .ino

Installation

  1. Download the file NotepadPlusPlus-4.1.nvda-addon
  2. Double-click it from Windows Explorer
  3. NVDA shows a confirmation dialog — accept it
  4. Restart NVDA (Ctrl+Alt+N or from the NVDA menu)

Uninstallation

  1. Open the NVDA menu
  2. Go to Tools > Add-on Manager
  3. Select the NotepadPlusPlus add-on
  4. Click "Remove"
  5. Restart NVDA

Language Detection

When opening a file in Notepad++, the module automatically detects the language from the file extension and announces it.

  • Python: announces "Python" for .py .pyw .pyi files
  • C/C++: announces "C/C++" for .c .cpp .cxx .h .hpp .ino files
  • Unknown: announces "type not recognized" for other files

Navigation shortcuts automatically adapt to the detected language.

Shortcuts — Information

F4
Announces the current line number. Works for all languages.
Ctrl+F4
Announces the indentation level of the current line.
  • Python: reports an error if indentation is not a multiple of 4.
  • C/C++: announces the number of spaces without judgment.

Shortcuts — Code Structure Navigation

NVDA+F7
Opens the navigation dialog (see dedicated section below).
F7 / Shift+F7
Go to next / previous class declaration. Announces: "class_name line X". Available in Python and C/C++.
F8 / Shift+F8
Go to next / previous function or method declaration. Announces: "function_name line X". Available in Python and C/C++.
F2
Go to the if __name__ == '__main__' block. Searches from the beginning of the file. Python only.

Navigation Dialog — NVDA+F7

The navigation dialog lists all functions and classes in the current file. It is inspired by the NVDA elements list available in browse mode.

On opening

  • Focus is placed on the filter field.
  • The list is pre-selected on the element closest to the current cursor position in the file.

Dialog contents

Filter field
Type to narrow the list by element name. Example: typing "init" shows only elements whose name contains "init" (__init__, initialize...). The filter applies in real time.
Radio buttons: All / Functions / Classes
Filter the list by element type. NVDA announces the number of elements when focus enters the list after a filter or type change.
Element list
Each element shows its name followed by its line number. Navigate with the up and down arrow keys. NVDA reads each element during navigation.
"Go to" button
Closes the dialog and positions the cursor on the line of the selected element in Notepad++. NVDA announces the line number.
"Close" button
Closes the dialog without navigating. The cursor stays in place.

Shortcuts — Indentation Navigation

These shortcuts work for all languages.

Alt+Down
Go to the next more-indented line. Announces: "text line X, indentation Y".
Alt+Up
Go to the next less-indented line. Announces: "text line X, indentation Y".
Ctrl+Alt+Down
Next line at the same indentation level. Announces: "text line X".
Ctrl+Alt+Up
Previous line at the same indentation level. Announces: "text line X".
Alt+Home
First line at the current indentation level. Announces: "text line X".
Alt+End
Last line at the current indentation level. Announces: "text line X".

Shortcuts — Block Selection

Ctrl+R
Select the current function or method.
Ctrl+Shift+R
Select the current class.
Shift+Alt+Down
Extend selection to the next indentation level change.
Shift+Alt+Up
Extend selection to the previous indentation level change.

Tip: after selecting a block with Ctrl+R or Ctrl+Shift+R, press the Delete key to remove the block.

Shortcuts — Code Execution

Ctrl+F5
Run the Python code in an external terminal (Windows command prompt). Python only.
