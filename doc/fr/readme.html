Presentation

Ce module ameliore l'accessibilite de l'editeur Notepad++ pour les developpeurs aveugles ou malvoyants utilisant le lecteur d'ecran NVDA. Il ajoute des raccourcis clavier pour naviguer dans le code source Python et C/C++ sans avoir a parcourir le fichier ligne par ligne.

Compatibilite :

  • Notepad++ 32 bits et 64 bits (Windows 10 / Windows 11)
  • NVDA 2025.1 minimum, teste jusqu'a NVDA 2026.1
  • Langages supportes :
    • Python : .py .pyw .pyi
    • C/C++/Arduino : .c .cpp .cxx .h .hpp .ino

Installation

  1. Telecharger le fichier NotepadPlusPlus-4.1.nvda-addon
  2. Double-cliquer dessus depuis l'Explorateur Windows
  3. NVDA affiche une boite de dialogue de confirmation — accepter
  4. Redemarrer NVDA (Ctrl+Alt+N ou depuis le menu NVDA)

Desinstallation

  1. Ouvrir le menu NVDA
  2. Aller dans Outils > Gestionnaire d'extensions
  3. Selectionner l'addon NotepadPlusPlus
  4. Cliquer sur "Supprimer"
  5. Redemarrer NVDA

Detection automatique du langage

A l'ouverture d'un fichier dans Notepad++, le module detecte automatiquement le langage a partir de l'extension du fichier et l'annonce vocalement.

  • Python : annonce "Python" pour les fichiers .py .pyw .pyi
  • C/C++ : annonce "C/C++" pour les fichiers .c .cpp .cxx .h .hpp .ino
  • Inconnu : annonce "type non reconnu" pour les autres fichiers

Les raccourcis de navigation s'adaptent automatiquement au langage detecte.

Raccourcis — Information

F4
Annonce le numero de la ligne courante. Fonctionne pour tous les langages.
Ctrl+F4
Annonce le niveau d'indentation de la ligne courante.
  • Python : signale une erreur si l'indentation n'est pas un multiple de 4.
  • C/C++ : annonce le nombre d'espaces sans jugement.

Raccourcis — Navigation par structure de code

NVDA+F7
Ouvre la fenetre de navigation (voir section dediee ci-dessous).
F7 / Maj+F7
Aller a la declaration de classe suivante / precedente. Annonce : "nom_classe ligne X". Disponible en Python et C/C++.
F8 / Maj+F8
Aller a la declaration de fonction ou methode suivante / precedente. Annonce : "nom_fonction ligne X". Disponible en Python et C/C++.
F2
Aller au bloc if __name__ == '__main__'. Recherche depuis le debut du fichier. Python uniquement.

Fenetre de navigation — NVDA+F7

La fenetre de navigation liste toutes les fonctions et classes du fichier courant. Elle s'inspire de la liste d'elements NVDA disponible en mode navigation web.

A l'ouverture

  • Le focus est place sur le champ de filtre.
  • La liste est pre-selectionnee sur l'element le plus proche de la position courante du curseur dans le fichier.

Contenu de la fenetre

Champ filtre
Tapez pour restreindre la liste par nom d'element. Exemple : taper "init" affiche uniquement les elements dont le nom contient "init" (__init__, initialize...). Le filtre s'applique en temps reel.
Boutons radio : Tout / Fonctions / Classes
Filtrer la liste par type d'element. NVDA annonce le nombre d'elements quand le focus entre dans la liste apres un changement de filtre ou de type.
Liste des elements
Chaque element affiche son nom suivi de son numero de ligne. Navigation avec les fleches haut et bas. NVDA lit chaque element lors de la navigation.
Bouton "Aller a"
Ferme la fenetre et positionne le curseur sur la ligne de l'element selectionne dans Notepad++. NVDA annonce le numero de ligne.
Bouton "Fermer"
Ferme la fenetre sans naviguer. Le curseur reste en place.

Raccourcis — Navigation par indentation

Ces raccourcis fonctionnent pour tous les langages.

Alt+Bas
Aller vers la prochaine ligne plus indentee. Annonce : "texte ligne X, indentation Y".
Alt+Haut
Aller vers la prochaine ligne moins indentee. Annonce : "texte ligne X, indentation Y".
Ctrl+Alt+Bas
Prochaine ligne de meme indentation. Annonce : "texte ligne X".
Ctrl+Alt+Haut
Ligne precedente de meme indentation. Annonce : "texte ligne X".
Alt+Debut
Premiere ligne du niveau d'indentation courant. Annonce : "texte ligne X".
Alt+Fin
Derniere ligne du niveau d'indentation courant. Annonce : "texte ligne X".

Raccourcis — Selection de blocs

Ctrl+R
Selectionner la fonction ou methode courante.
Ctrl+Maj+R
Selectionner la classe courante.
Maj+Alt+Bas
Etendre la selection jusqu'au prochain changement de niveau d'indentation.
Maj+Alt+Haut
Etendre la selection jusqu'au changement de niveau precedent.

Astuce : apres avoir selectionne un bloc avec Ctrl+R ou Ctrl+Maj+R, appuyez sur la touche Suppression du clavier pour supprimer le bloc.

Raccourcis — Execution du code

Ctrl+F5
Executer le code Python dans un terminal externe (invite de commandes Windows). Python uniquement.
